Introduction

Tic-Tac-Toe is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3×3 grid. The player who succeeds in placing three respective marks in a horizontal, vertical, or diagonal row wins the game.

$ git clone git@github.com:slovey528/Tic-Tac-Toe.git

Main Features

  • Supports customizing player name
  • Pop-up scoreboard for current players
  • Utilized Model-View-Controller (MVC) design pattern
  • More features developing (N×N board, single player mode with difficulty level)...
